# Package Manager Updater

CLI tool to update all package managers and check the latest update date 
![Linux](https://img.shields.io/badge/Linux-FCC624?style=for-the-badge&logo=linux&logoColor=black) ![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
---

## Funcs

### `--update`  
Updates all packages from All Package Managers

**List of package managers:**
- Pacman
- Flatpak
- Dandified YUM [DNF]
- Snap
- Yay
- PIP
- NPM

### `--latest`
Shows the latest update
